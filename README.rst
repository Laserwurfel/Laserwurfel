Laserwürfel
===========

Ein 3D-Puzzlespiel für Windows, Mac OS und Linux.

Dependencies
============

GLFW library
    http://www.glfw.org/download.html

GLFW Python bindings
    ``pip install glfw``

PyAudio
    https://people.csail.mit.edu/hubert/pyaudio/#downloads

oggdec
    ``vorbis-tools`` or http://rarewares.org/ogg-oggdec.php
